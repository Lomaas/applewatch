# Configure TableView

1. Lag noe fake data for å teste presentasjonen av dataen 
2. Sett antall rader og konfigurer hver rad med metodene nedenfor

```swift
tableView.setNumberOfRows..
let row = tableView.rowControllerAtIndex...
```
